# 编程实践索引目录

编程实践中，经常会遇到问题却不知该搜索什么关键字。

本项目提供了编程实践中常见的技术问题的关键字索引及说明。

本项目不追求大而全（我们有Wiki），而是用最简洁直观的方式，告诉你“是什么”、以及“有些什么”。

这样，当遇到问题时，就有了搜索方向。



本项目应作为索引使用。首先他不可能囊括所有的知识点，其次其中一定会有错误，找到关键字后，你还需要进行二次搜索，以获得更多更准确的信息。

若发现错误，或没有找到你想要的信息，欢迎反馈。
